# Azeroth Pilot Reloaded Pro - Texture Assets

This directory contains custom textures for the STUNNING Modern UI.

## Required Textures:

### Backgrounds
- `GradientBG.tga` - Smooth gradient background for main window
- `HeaderGradient.tga` - Header gradient overlay
- `CardBG.tga` - Card background texture

### Effects
- `Shimmer.tga` - Animated shimmer effect for progress bars
- `Glow.tga` - Soft glow effect for highlights
- `Particle.tga` - Particle effects for animations

### Icons
- `APR_Logo.tga` - Main addon logo (256x256)
- `APR_Icon.tga` - Minimap button icon (64x64)
- `Waypoint.tga` - Custom waypoint marker

## Texture Specifications:

All textures should be:
- Format: TGA or BLP (BLP preferred for performance)
- Power-of-2 dimensions (64, 128, 256, 512, etc.)
- No alpha channel unless transparency needed
- Compressed for optimal file size

## Creating Textures:

You can use tools like:
- GIMP (free)
- Photoshop
- BLP Converter for WoW-specific format

## Fallback:

If custom textures are not available, the UI will use WoW's built-in textures as fallbacks.

**Note**: These placeholder files ensure the UI works even without custom graphics.
